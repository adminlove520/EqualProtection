## 注意事项
- 所有脚本在使用前应当注意查看代码，防止存在如sshd配置root不可远程登录（无法使用root通过ssh22访问）、启用iptables（主要会对sshd22端口deny从而导致shell断开无法连接）、passwd等文件权限（无法创建账号或使用passwd -l等命令）。
- 对于[【基线 加固资料&工具】](./基线%20%20加固资料&工具/*)中文件应当注意是否适用于自己的项目或云资源。
